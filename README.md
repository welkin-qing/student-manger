# login
1. 获取表单数据
2. 根据表单中不同身份查询对应数据表
  - 判断用户账号是否存在
  - 用户账号存在，密码是否正确
3. 发送响应数据，使用session记录用户身份

# admin
1. 使用bootstrapValidator
  - 动态处理表单输入是否为空
  - 动态处理新密码和确认密码输入是否相同
2. 判断原始密码和输入的当前密码是否一致，若不一致则返回错误提示
3. 更新数据库当前密码，并重新登录

# profile
1. 根据不同身份展示不同个人信息页面信息词条
2. 更新数据用户身份信息，并刷新页面

# new
1. 该页面上传文件时，使用七牛云处理文件上传
2. 点击上传文件时，使用弹出框进行提示
3. 设计file表，将发布数据保存在file表中

# index
1. 完成数据库到该页面的渲染
2. 对数据进行时间显示，内容显示
3. 对不同发布内容进行不同显示，包括（图片，下载）
# 其他
1. 处理进入页面，input标签自动focus
2. 提交非空数据，前端对数据进行处理
